# Bộ số may mắn

Trang web tĩnh tạo 5 số ngẫu nhiên không trùng nhau từ 01 đến 55.

## Chạy local

Mở `index.html` trực tiếp hoặc chạy:

```bash
python3 -m http.server 8000
```

Sau đó truy cập `http://localhost:8000`.

## Công nghệ

- HTML
- CSS
- JavaScript thuần
- Web Crypto API cho nguồn số ngẫu nhiên
